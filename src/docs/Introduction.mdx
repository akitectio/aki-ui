# Introduction to Aki UI

Aki UI is a modern React component library designed for building beautiful and accessible user interfaces with a focus on developer experience and performance.

## Getting Started

To use Aki UI in your project, install it via npm:

```bash
npm install @akitectio/aki-ui
```

Then import the components you need:

```jsx
import { Button, Card, FormControl } from '@akitectio/aki-ui';
import '@akitectio/aki-ui/css';

function App() {
  return (
    <div>
      <Card>
        <Card.Header>Welcome to Aki UI</Card.Header>
        <Card.Body>
          <p>Get started with our beautiful components</p>
          <Button variant="primary">Get Started</Button>
        </Card.Body>
      </Card>
    </div>
  );
}
```

## Key Features

- **Modern Design**: Clean, minimalist design that looks great in any application
- **Accessibility**: Built with accessibility in mind from the ground up
- **TypeScript Support**: Full TypeScript support with comprehensive type definitions
- **Customizable**: Easy to customize with Tailwind CSS
- **Lightweight**: Optimized bundle size for better performance
